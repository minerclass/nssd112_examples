# NSSD112 Board Meeting Dashboard

This folder contains a standalone, browser-based dashboard for North Shore School District 112 Board of Education meeting information for the 2025-26 school year.

## Contents

- `index.html` - the complete dashboard, including meeting timeline, category filters, search, meeting cards, public links, and local notes.

## How to Preview

Open `index.html` in a browser. No server or build step is required.

## Privacy Notes

The dashboard is static. It does not send searches, notes, or pasted agenda text to a server.

Meeting notes are saved only in the browser's local storage on the machine where the file is opened. Avoid pasting confidential student, staff, legal, personnel, or closed-session information into the notes field.

## Review Notes

Board meeting content should be checked against the district's official agenda, minutes, BoardBook records, and livestream links before public use.
