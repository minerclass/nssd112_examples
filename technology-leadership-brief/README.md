# NSSD112 Technology Leadership Brief

This is a standalone local interview and reference tool for the North Shore School District 112 Director of Technology and Innovation interview process.

## Files

- `index.html` - Interactive technology leadership brief with tabbed sections, accordions, comparison language, and a print/export brief.

## How to Use

Open `index.html` in a browser.

Use the tabs to rehearse different parts of the interview:

- Brief
- Alignment
- WIGs
- Migration
- AI
- Parallel Practice
- Questions
- Print Brief

Use the browser print command or the built-in "Print or Save as PDF" button to export the brief.

## Privacy and Sharing Notes

This folder has not been added to the existing GitHub repository.

Before sharing externally, verify all district facts, names, dates, recognitions, report card data, and claims against official sources such as:

- NSSD112 public website
- Illinois Report Card
- NSSD112 Board packets and minutes
- Official CoSN/TLE announcements
- Your own verified BPD3 artifacts

The "Parallel Practice" section is intentionally framed as professional comparison and collegial context, including Lake County, CoSN, and conference-related professional overlap where appropriate.
